This repository contains my submission for Task 3 under the Data Science & Analytics track (DS) by Future Interns.

## 📌 Objective
The objective of this project is to analyze marketing funnel data to understand user behavior across different stages (Visitors → Leads → Customers) and identify opportunities to improve conversion rates.

## 🔍 Key Analysis Performed
- Traffic-to-Lead Conversion Analysis
- Lead-to-Customer Conversion Analysis
- Funnel drop-off analysis at each stage
- Channel / campaign performance comparison
- Time-based conversion trends

## 📊 Insights
- Identified stages where maximum user drop-off occurs
- Analyzed which channels generate high-quality leads
- Observed conversion performance across different campaigns
- Highlighted key areas affecting overall funnel efficiency

## 💡 Recommendations
- Optimize high drop-off stages in the funnel
- Focus on high-performing marketing channels
- Improve lead nurturing strategies
- Enhance targeting and campaign effectiveness
